```
https://mercatiwdg.ilsole24ore.com/FinanzaMercati/api/TimeSeries/GetTimeSeries/IT0005217770.MOT?timeWindow=OneDayCurrent&
```

Time parameters:
- OneDayCurrent
- OneWeek
- OneMonth
- TreeMonths (Three senza h)
- SixMonths
- OneYear
- FiveYears
- TenYears
